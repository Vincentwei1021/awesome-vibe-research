# Contributing

Thanks for helping improve `awesome-vibe-research`.

## What to Add

This list is for tools, platforms, protocols, and references that are meaningfully useful for vibe research, including:

- literature discovery and review
- research-agent systems and orchestration
- experiment execution and compute infrastructure
- reproducibility, evaluation, and reporting
- methodologies, standards, and reference material

## Submission Guidelines

- Add a short, concrete description of what the project does.
- Explain why it belongs in this list and which layer of the research stack it supports.
- Prefer primary sources such as the official repository, project site, or documentation.
- Append new entries to the end of the relevant category instead of inserting them at the top.
- Keep descriptions concise and neutral.
- Do not add tools that are too generic unless they are clearly useful in research workflows.
- Avoid dead projects, obvious spam, or tools with unclear relevance to vibe research.

## Formatting

- Keep entry format consistent with the rest of the list.
- Do not rename projects unless the official branding has changed.
- Preserve existing category structure unless there is a strong reason to introduce a new section.
- If a project fits multiple categories, pick the most useful primary category rather than duplicating it.

## Translations

- The English README is the source of truth.
- Translation files may lag behind slightly.
- If you update translated files, keep section order and links aligned with the English version.
